# Простой калькулятор на C#

## Описание
Калькулятор с базовыми операциями, созданный двумя разработчиками за 1 час.

## Функции
- Сложение, вычитание, умножение, деление
- Очистка (C), удаление символа (←)
- Проценты (%)
- Работа с десятичными числами
- Ввод с клавиатуры
- История операций
