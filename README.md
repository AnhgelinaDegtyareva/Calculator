# Простой калькулятор на C#

## Описание
Калькулятор с базовыми операциями, созданный двумя разработчиками.

## Функции
- Сложение, вычитание, умножение, деление
- Очистка (C), удаление символа (←)
- Проценты (%)
- История операций
